# RBN Analyzer

Веб-приложение для анализа данных **Reverse Beacon Network** — статистика и визуализация спотов по диапазонам, странам, позывным и времени.

## Возможности

- **Анализ по диапазонам** — 160m, 80m, 60m, 40m, 30m, 20m, 17m, 15m, 12m, 10m
- **Мультиязычность** — русский / английский интерфейс
- **10 графиков**: топ-20 стран, топ-20 позывных, активность по часам, дням недели, месяцам, сезонам, heatmaps (день×час, месяц×час, страна×час, страна×день)
- **Полные названия стран** — 1105 маппингов префиксов (RU/EN)
- **Гибкий период** — кнопки: месяц, 3 месяца, полгода, год
- **Тёмный интерфейс** — адаптивный дизайн

## Запуск локально

```bash
pip install -r requirements.txt
python3 app.py
```

Откройте http://localhost:5000

## Деплой на Render

1. Загрузите код на GitHub
2. На [render.com](https://render.com): New → Web Service → подключите репозиторий
3. Render автоматически подхватит `render.yaml`

## Структура

```
rbn_analyzer/
├── app.py              # Flask веб-приложение
├── analyzer.py         # Ядро анализа и генерации графиков
├── prefix_data.py      # Маппинг префиксов → страны (1105 записей)
├── templates/
│   └── index.html      # Интерфейс (мультиязычный)
├── requirements.txt    # Зависимости
├── render.yaml         # Конфиг для Render
├── rbn_analyzer_20m.py # CLI скрипт (оригинал)
└── what.md             # Описание CLI скрипта
```
