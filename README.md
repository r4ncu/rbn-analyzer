# RBN Analyzer

Веб-приложение для анализа данных **Reverse Beacon Network** — статистика и визуализация спотов по диапазонам, странам, позывным и времени.

## Возможности

- **Анализ по диапазонам** — 160m, 80m, 60m, 40m, 30m, 20m, 17m, 15m, 12m, 10m
- **Мультиязычность** — русский / английский интерфейс
- **10 графиков**: топ-20 стран, топ-20 позывных, активность по часам, дням недели, месяцам, сезонам, heatmaps (день×час, месяц×час, страна×час, страна×день)
- **Полные названия стран** — 1105 маппингов префиксов (RU/EN)
- **Гибкий период** — кнопки: месяц, 3 месяца, полгода, год

## Запуск локально

### Требования

- Python 3.9+
- pip

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/r4ncu/rbn-analyzer.git
cd rbn-analyzer

# Установите зависимости
pip install -r requirements.txt

# Запустите
python3 app.py
```

Откройте http://localhost:5000 в браузере.

### Альтернативно — CLI скрипт

Если не нужен веб-интерфейс, есть отдельный скрипт для генерации графиков:

```bash
python3 rbn_analyzer_20m.py
```

Результаты сохранятся в папку `rbn_results_20m/`.

## Почему не работает на Render.com

Приложение **не предназначено** для деплоя на бесплатный тариф Render.com (или аналогичные free-tier платформы) по следующим причинам:

1. **Объём данных** — анализ за год требует скачивания ~1.3 GB архивов RBN (365 файлов по ~3.5 MB). Бесплатные платформы имеют ограничения на время выполнения и память.

2. **Оперативная память** — даже с оптимизациями, обработка 365 дней данных с использованием pandas, matplotlib и seaborn требует больше 512 MB RAM, что является лимитом бесплатного тарифа Render.

3. **Время выполнения** — скачивание и обработка занимают несколько минут, что превышает типичные таймауты бесплатных платформ.

4. **Нестабильность** — бесплатные инстансы «засыпают» после бездействия и перезапускаются при обращении, что прерывает длительные процессы анализа.

Для работы приложения требуется **локальный компьютер** или выделенный сервер.

## Структура проекта

```

├── app.py              # Flask веб-приложение
├── analyzer.py         # Ядро анализа и генерации графиков
├── prefix_data.py      # Маппинг префиксов → страны (1105 записей)
├── templates/
│   └── index.html      # Интерфейс (мультиязычный)
├── requirements.txt    # Зависимости
├── rbn_analyzer_20m.py # CLI скрипт
├── post.md             # Пост-анонс
└── what.md             # Описание CLI скрипта
```

## Технологии
