# Как запустить RBN Analyzer — простая инструкция

## Шаг 1: Установи Python

Если у тебя macOS — Python уже есть. Проверь:

Открой **Терминал** (найди через Spotlight, введи `Terminal`) и набери:

```
python3 --version
```

Если видишь что-то вроде `Python 3.11.x` — всё ок, иди в Шаг 2.

Если пишет `command not found` — скачай Python с https://www.python.org/downloads/ и установи.

## Шаг 2: Скачай программу

В Терминале набери по очереди (каждую строку — отдельно, жми Enter после каждой):

```
cd ~/Desktop
```

```
git clone https://github.com/r4ncu/rbn-analyzer.git
```

```
cd rbn-analyzer
```

## Шаг 3: Установи нужные библиотеки

Набери:

```
pip3 install -r requirements.txt
```

Подожди пока установится. Может занять 1-2 минуты.

## Шаг 4: Запусти

Набери:

```
python3 app.py
```

Увидишь что-то вроде:

```
 * Running on http://127.0.0.1:5000
```

## Шаг 5: Открой в браузере

Открой браузер (Chrome, Safari, Firefox) и перейди по адресу:

```
http://localhost:5000
```

Готово! Можно пользоваться.

## Как пользоваться

1. В поле **Позывной** введи свой позывной (например `R4NCU`)
2. Выбери **диапазон** (по умолчанию 20m)
3. Выбери **период** (год, полгода, 3 месяца или месяц)
4. Нажми **Запустить анализ**
5. Жди — сначала загрузка данных (1-3 минуты), потом появятся графики

## Как остановить

В Терминале нажми `Ctrl+C`

## Частые проблемы

**"pip3: command not found"**
Попробуй вместо `pip3` написать `pip`:
```
pip install -r requirements.txt
```

**"Permission denied"**
Добавь `--user`:
```
pip3 install --user -r requirements.txt
```

**"Port 5000 already in use"**
Кто-то уже запустил программу на этом порту. Закрой предыдущее окно Терминала с запущенным `app.py` или перезагрузи компьютер.
