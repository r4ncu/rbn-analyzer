# what
**Скрипт на Python rbn_analyzer_20m.py**

выполняет сбор и расчет статистики диапазона 20 метров для моего споттера на reversebeacon.net. Используется многопоточность для быстрой загрузки ежедневных архивов RBN за последние 365 дней.

* Фильтрация: автоматически извлекает данные и фильтрует их по позывному споттера (в данном случае R4NCU).
* Анализ: проводит статистическую обработку по странам, позывным, времени суток, месяцам, дням недели и сезонам.
* Визуализация: создает графики и диаграммы.
* Сохранение: все результаты (графики, таблицы и полная база данных) сохраняются в отдельную папку rbn_results.

Для запуска скрипта потребуются установленные библиотеки pandas, requests, matplotlib и seaborn. Устанавливаются командой:
pip install pandas requests matplotlib seaborn
В моём случае под macOS запускается командой python3 /Users/bass/Desktop/radio/rbn/rbn_analyzer_20m.py
